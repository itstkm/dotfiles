# WezTerm 操作マニュアル

リーダーキー: `Ctrl+q`（押して離してから次のキーを押す。2秒以内）

## タブ操作

### タブの作成・削除

| キー | 動作 |
|------|------|
| `Cmd+t` | 新しいタブを作成 |
| `Cmd+w` | タブを閉じる（確認あり） |

### タブの切り替え

| キー | 動作 |
|------|------|
| `Ctrl+Tab` | 次のタブへ |
| `Ctrl+Shift+Tab` | 前のタブへ |
| `Cmd+1~8` | 番号で直接切り替え |
| `Cmd+9` | 最後のタブへ |

### タブの順序入れ替え

| キー | 動作 |
|------|------|
| `Ctrl+q` → `}` | 今のタブを右に移動 |
| `Ctrl+q` → `{` | 今のタブを左に移動 |

## ペイン操作

### ペインの作成

| キー | 動作 |
|------|------|
| `Ctrl+q` → `d` | 縦に分割（上下にペインが増える） |
| `Ctrl+q` → `r` | 横に分割（左右にペインが増える） |

### ペインの削除

| キー | 動作 |
|------|------|
| `Ctrl+q` → `x` | 今いるペインを閉じる（確認あり） |

### ペイン間の移動

| キー | 動作 |
|------|------|
| `Ctrl+q` → `h` | 左のペインへ |
| `Ctrl+q` → `j` | 下のペインへ |
| `Ctrl+q` → `k` | 上のペインへ |
| `Ctrl+q` → `l` | 右のペインへ |

### ペインのサイズ変更

| キー | 動作 |
|------|------|
| `Ctrl+q` → `s` | リサイズモードに入る |
| リサイズモード中 `h/j/k/l` | 左/下/上/右に1ずつ広げる |
| リサイズモード中 `Enter` | リサイズモードを抜ける |

### その他

| キー | 動作 |
|------|------|
| `Ctrl+q` → `z` | 今いるペインを全画面表示（もう一度押すと戻る） |
| `Ctrl+Shift+[` | ペイン選択UI（各ペインに番号が表示される） |

## ワークスペース（タブのグループ管理）

| キー | 動作 |
|------|------|
| `Ctrl+q` → `w` | ワークスペース一覧から切り替え |
| `Ctrl+q` → `W` | 新規ワークスペースを作成 |
| `Ctrl+q` → `$` | ワークスペース名を変更 |

## コピー・ペースト

| キー | 動作 |
|------|------|
| `Cmd+c` | コピー |
| `Cmd+v` | ペースト |
| `Ctrl+q` → `[` | コピーモード（vim風の範囲選択） |

## その他

| キー | 動作 |
|------|------|
| `Alt+Enter` | フルスクリーン切替 |
| `Cmd+p` | コマンドパレット |
| `Ctrl+Shift+r` | 設定再読み込み |
| `Ctrl + +/-/0` | フォントサイズ変更/リセット |
