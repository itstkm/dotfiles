# Neovim (LazyVim) 操作マニュアル

リーダーキー: `Space`

## 基本操作

### ファイル操作

| キー | 動作 |
|------|------|
| `Space + f + f` | ファイル検索 |
| `Space + s + g` | 全文検索（grep） |
| `Cmd+s` | 保存 |
| `Cmd+z` | 元に戻す |
| `Cmd+Shift+z` | やり直し |

### ファイルツリー（neo-tree）

| キー | 動作 |
|------|------|
| `Space + e` | ファイルツリーの開閉 |
| `Ctrl+w + h` | ファイルツリーにフォーカス |
| `Ctrl+w + l` | エディタにフォーカスを戻す |
| `j / k` | 上下に移動（ツリー内） |
| `Enter` | ファイルを開く / フォルダを展開 |
| `a` | ファイル・フォルダ作成 |
| `d` | 削除 |
| `r` | 名前変更 |
| `<` / `>` | Files / Git / Buffers タブ切り替え |

### VSCode風ショートカット

| キー | 動作 |
|------|------|
| `Cmd+p` | ファイル検索 |
| `Cmd+Shift+f` | 全文検索 |
| `Cmd+b` | サイドバー開閉 |
| `Cmd+/` | コメントトグル |

## Git操作

| キー | 動作 |
|------|------|
| `Space + g + g` | lazygit を開く |
| `Space + g + b` | git blame（行ごとの変更者） |
| `Space + g + d` | diff表示 |
| `Space + g + e` | Git変更ファイル一覧（neo-tree） |

## バッファ（開いているファイル）

| キー | 動作 |
|------|------|
| `Space + b + e` | バッファ一覧（neo-tree） |
| `Space + b + d` | 現在のバッファを閉じる |

## コピー（ファイル全体）

1. `gg` — ファイル先頭に移動
2. `V` — 行選択モードに入る
3. `G` — ファイル末尾まで選択
4. `y` — コピー

## プラグイン管理

| キー | 動作 |
|------|------|
| ダッシュボードで `l` | lazy.nvim管理画面 |
| ダッシュボードで `x` | Lazy Extras（言語追加など） |
